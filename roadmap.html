Roadmap

Planned growth

A staged plan for implementation, validation, and community release.

Phase 1 — Core implementation (current)

Build and harden plate reader parsing, feature generation, and statistical analysis foundations, with initial validation on E. coli O157:H7 datasets.

Phase 2 — Validation and documentation (next)

Expand analysis across published datasets, improve reproducible documentation, and provide clear tutorial notebooks and benchmarking references.

Phase 3 — Community release (future)

Support additional plate reader adapters, richer notebook workflows, example datasets with attribution, and broader API-level usability.
